# Campaign-Dashboard-Powerbi

Admissions Campaign Dashboard | Power BI

Built an end to end Power BI dashboard to analyze admissions campaign performance from lead generation to final enrollment.

Designed KPIs including total leads, admissions count, and lead to admission conversion rate.

Performed campaign wise and branch wise performance analysis to identify high impact marketing channels.

Used DAX to create calculated measures for conversion rate, trend analysis, and performance comparison.

Implemented interactive filters and slicers to enable dynamic data exploration.

Delivered actionable insights to support data driven marketing and admissions decisions.
